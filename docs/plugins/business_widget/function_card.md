# 功能卡片

## 简介
横向布局的卡片，从左到右是：icon - 主标题 - 副标题 - 右侧内容。
使用spaceBetween对其，icon - 主标题 - 副标题在最左侧，右侧内容在最右侧。

## 属性

| 参数name   | 类型     | 必传    | 默认值  | 描述     |
|----------|--------|-------|------|--------|
| icon     | Widget | false | null | icon内容 |
| title    | String | true  | -    | 主标题    |
| subTitle | String | false | null | 副标题    |
| child    | Widget | true  | -    | 右侧内容   |
